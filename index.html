E-commerce Flipkart Clone

A responsive e-commerce web application based on Flipkart built with React.js and hosted on Netlify.

    Key features :

  • User Authentication - Login & Signup process with proper validation & secure routes.
  • Product Details Page - Individual product page with ratings, description, and "Add to Cart" button.
  • Shopping Cart & Wishlist - Remove/add products from cart and favorites using React Context API for state management.
  • User Profile Section - user vise data saved in localstorage and after login show all data of user in profile.
  • Notifications System - Real-time confirmation (login success, order placed, account created) with custom components.
  • Responsive UI - Mobile-first approach, optimized for desktop.

Tech Stack :

  • Frontend - React.js (with Vite), React Router Dom, Context API, Axios
  • Styling - Tailwind CSS
  • State Management - Context API
  • Build Tool - Vite
  • Live Website : https://e-commerce-msd-25.netlify.app/
  • Source Code : https://github.com/Sahevagmeniya/E-commerce_Flipkart_Clone
